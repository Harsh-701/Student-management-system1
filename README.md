**STUDENT MANAGEMENT SYSTEM**

A Full stack web application built using ASP.NET Core Razor Pages to manage student records with complete CRUD functionality.

****FEATURES****

*View student details 

*Edit student details

*Delete student details

*Data stored in Sql server using Entity Framework Core.

**TECHNOLOGIES USED**

*Asp.Net Core Razor Pages

*C#

*Entity Framework Core

*SQL Server(LocalDB)

HTML,CSS,Bootstrap
